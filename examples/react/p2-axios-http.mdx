This example demonstrates how to work with HTTP methods (GET and POST) in React and set up client-side routing with React Router v7.

## Overview

You'll learn how to:

- Set up React Router for client-side navigation
- Fetch data with the Fetch API (GET requests)
- Submit data with POST requests
- Manage component state with React hooks
- Handle form submissions

## Router Configuration

First, set up React Router in your `App.tsx`:

```tsx src/App.tsx
import { BrowserRouter, Routes, Route } from "react-router-dom";
import MetodoGet from "./pages/MetodoGet";
import MetodoPost from "./pages/MetodoPost";

function App() {
  return (
    <BrowserRouter>
      <Routes>
        <Route path="/" element={<MetodoGet />} />
        <Route path="/get" element={<MetodoGet />} />
        <Route path="/post" element={<MetodoPost />} />
      </Routes>
    </BrowserRouter>
  )
}

export default App
```

<Note>
  This configuration creates three routes: the home page and `/get` both show the GET example, while `/post` shows the POST example.
</Note>

## HTTP GET Example

The `MetodoGet` component fetches data from the Pokemon API:

```tsx src/pages/MetodoGet.tsx
import { useEffect, useState } from "react";

const url = "https://pokeapi.co/api/v2/pokemon?limit=3000";

const MetodoGet = () => {
    const [data, setData] = useState(null);

    useEffect(() => {
        fetch(url)
            .then((res) => res.json())
            .then((data) => setData(data));
    }, []);

    return (
        <>
            <ul>
                {data?.results?.map((item) => {
                    return <li>
                        <a href={item.url}>{item.name}</a>
                    </li>;
                })}
            </ul>
        </>
    )
}

export default MetodoGet;
```

### GET Request Breakdown

<Tabs>
  <Tab title="State Management">
    ```tsx
    const [data, setData] = useState(null);
    ```

    - Uses `useState` to store the API response
    - Initializes with `null` before data loads
    - `setData` updates the state when data arrives
  </Tab>

  <Tab title="Data Fetching">
    ```tsx
    useEffect(() => {
        fetch(url)
            .then((res) => res.json())
            .then((data) => setData(data));
    }, []);
    ```

    - `useEffect` runs when component mounts
    - Empty dependency array `[]` ensures it runs once
    - Fetch API makes the GET request
    - Response is parsed as JSON and stored in state
  </Tab>

  <Tab title="Rendering Data">
    ```tsx
    {data?.results?.map((item) => {
        return <li>
            <a href={item.url}>{item.name}</a>
        </li>;
    })}
    ```

    - Optional chaining `?.` safely accesses nested properties
    - `map` iterates over the results array
    - Each Pokemon is rendered as a list item with a link
  </Tab>
</Tabs>

## HTTP POST Example

The `MetodoPost` component handles form submission and POST requests:

```tsx src/pages/MetodoPost.tsx
import { useState } from 'react';

const MetodoPost = () => {
  const [titulo, setTitulo] = useState('');
  const [cuerpo, setCuerpo] = useState('');
  const [respuesta, setRespuesta] = useState(null);

  const handleSubmit = (e) => {
    e.preventDefault();

    fetch('https://jsonplaceholder.typicode.com/posts', {
      method: 'POST',
      body: JSON.stringify({
        title: titulo,
        body: cuerpo,
        userId: 1,
      }),
      headers: {
        'Content-type': 'application/json; charset=UTF-8',
      },
    })
      .then((response) => response.json())
      .then((json) => {
        setRespuesta(json);
        console.log('Post creado (simulado):', json);
      })
      .catch((error) => console.error('Error:', error));
  };

  return (
    <div>
      <h2>Crear nuevo post</h2>
      <form onSubmit={handleSubmit}>
        <input
          type="text"
          placeholder="Título"
          value={titulo}
          onChange={(e) => setTitulo(e.target.value)}
        />
        <textarea
          placeholder="Cuerpo del post"
          value={cuerpo}
          onChange={(e) => setCuerpo(e.target.value)}
        />
        <button type="submit">Enviar</button>
      </form>
      {respuesta && (
        <div>
          <h3>Respuesta de la API:</h3>
          <pre>{JSON.stringify(respuesta, null, 2)}</pre>
        </div>
      )}
    </div>
  );
};

export default MetodoPost;
```

### POST Request Breakdown

<Tabs>
  <Tab title="Form State">
    ```tsx
    const [titulo, setTitulo] = useState('');
    const [cuerpo, setCuerpo] = useState('');
    const [respuesta, setRespuesta] = useState(null);
    ```

    - `titulo`: Stores the post title
    - `cuerpo`: Stores the post body
    - `respuesta`: Stores the API response
  </Tab>

  <Tab title="Form Handling">
    ```tsx
    const handleSubmit = (e) => {
      e.preventDefault();
      // ... fetch logic
    };
    ```

    - Prevents default form submission behavior
    - Makes POST request to JSONPlaceholder API
    - Updates state with the response
  </Tab>

  <Tab title="POST Configuration">
    ```tsx
    fetch(url, {
      method: 'POST',
      body: JSON.stringify({
        title: titulo,
        body: cuerpo,
        userId: 1,
      }),
      headers: {
        'Content-type': 'application/json; charset=UTF-8',
      },
    })
    ```

    - Specifies POST method
    - Converts data to JSON string
    - Sets proper content-type header
  </Tab>

  <Tab title="Controlled Inputs">
    ```tsx
    <input
      value={titulo}
      onChange={(e) => setTitulo(e.target.value)}
    />
    ```

    - Input value is controlled by React state
    - `onChange` updates state on every keystroke
    - Ensures single source of truth
  </Tab>
</Tabs>

## Common Patterns

<AccordionGroup>
  <Accordion title="Error Handling">
    Add error handling to your fetch requests:

    ```tsx
    const [error, setError] = useState(null);

    fetch(url)
      .then(res => res.json())
      .then(data => setData(data))
      .catch(err => setError(err.message));
    ```
  </Accordion>

  <Accordion title="Loading States">
    Show loading indicators while data fetches:

    ```tsx
    const [loading, setLoading] = useState(true);

    useEffect(() => {
      setLoading(true);
      fetch(url)
        .then(res => res.json())
        .then(data => {
          setData(data);
          setLoading(false);
        });
    }, []);

    if (loading) return <p>Loading...</p>;
    ```
  </Accordion>

  <Accordion title="Using Axios Instead">
    While this example uses the Fetch API, you can also use Axios:

    ```bash
    npm install axios
    ```

    ```tsx
    import axios from 'axios';

    const response = await axios.get(url);
    setData(response.data);
    ```
  </Accordion>
</AccordionGroup>
