This example demonstrates how to create a React application with TypeScript, Vite, and Tailwind CSS v4 using the latest configuration approach.

## Overview

You'll learn how to:

- Initialize a Vite + React + TypeScript project
- Install and configure Tailwind CSS v4
- Use the new `@tailwindcss/vite` plugin
- Create styled components with Tailwind utilities

## Project Setup

<Steps>
  <Step title="Create Vite Project">
    Initialize a new Vite project with React and TypeScript:

    ```bash
    npm create vite@latest my-react-app -- --template react-ts
    cd my-react-app
    ```
  </Step>

  <Step title="Install Dependencies">
    Install Tailwind CSS v4 and the Vite plugin:

    ```bash
    npm install tailwindcss @tailwindcss/vite
    ```

    <Note>
      Tailwind CSS v4 uses a new Vite plugin approach instead of PostCSS configuration.
    </Note>
  </Step>

  <Step title="Configure Vite">
    Update your `vite.config.ts` to include the Tailwind plugin:

    ```typescript vite.config.ts
    import { defineConfig } from 'vite'
    import react from '@vitejs/plugin-react'
    import tailwindcss from '@tailwindcss/vite'

    export default defineConfig({
      plugins: [react(), tailwindcss()],
    })
    ```
  </Step>

  <Step title="Import Tailwind CSS">
    Update your `src/index.css` file to import Tailwind:

    ```css src/index.css
    @import "tailwindcss";
    ```

    This single line replaces the traditional three `@tailwind` directives.
  </Step>

  <Step title="Start Development Server">
    Run the development server:

    ```bash
    npm run dev
    ```

    Your app should now be running with Tailwind CSS fully configured!
  </Step>
</Steps>

```tsx src/App.tsx

          ¡Tailwind Configurado! 🚀
        </h1>
        <p className="text-gray-600 mb-6">

        </p>
        <div className="flex gap-4 justify-center">

            Vite
          </span>
        </div>
      </div>
    </div>
  )
}

export default App
```

## Styling Breakdown

<Tabs>
  <Tab title="Container">
    ```tsx
    <div className="min-h-screen bg-slate-900 flex flex-col items-center justify-center p-4">
    ```

    - `min-h-screen`: Full viewport height
    - `bg-slate-900`: Dark background
    - `flex flex-col`: Vertical flexbox layout
    - `items-center justify-center`: Center content
  </Tab>

  <Tab title="Card">
    ```tsx
    <div className="bg-white p-8 rounded-2xl shadow-2xl text-center hover:scale-105 transition-transform duration-300 border-4 border-indigo-500">
    ```

    - `bg-white`: White background
    - `rounded-2xl`: Large border radius
    - `shadow-2xl`: Large shadow
    - `hover:scale-105`: Scale up on hover
    - `transition-transform duration-300`: Smooth transition
  </Tab>

  <Tab title="Typography">
    ```tsx
    <h1 className="text-3xl font-bold text-indigo-600 mb-4">
    ```

    - `text-3xl`: Large text size
    - `font-bold`: Bold font weight
    - `text-indigo-600`: Indigo color
    - `mb-4`: Bottom margin
  </Tab>

  <Tab title="Badges">
    ```tsx
    <span className="px-4 py-2 bg-blue-100 text-blue-700 rounded-full text-sm font-semibold">
    ```

    - `px-4 py-2`: Horizontal and vertical padding
    - `bg-blue-100`: Light blue background
    - `rounded-full`: Fully rounded corners
    - `animate-bounce`: Built-in bounce animation
  </Tab>
</Tabs>
