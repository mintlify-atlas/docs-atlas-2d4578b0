## Course Structure

The course is organized into three main sections:

### 📚 Documentation & Glossary

Learn fundamental concepts and terminology used throughout web development. Our glossary covers JavaScript modern syntax, React concepts, and essential development tools.

- [JavaScript Concepts](/glossary/javascript-concepts) — Arrow functions, array methods, and modern syntax
- [React Concepts](/glossary/react-concepts) — JSX, components, and React fundamentals
- [Development Tools](/glossary/tools) — NVM, error handling, and tooling

### 💻 Practical Examples

Explore working code examples that demonstrate real-world patterns:

**React Examples:**
- Basic React + TypeScript + Vite setup with Tailwind CSS
- HTTP methods (GET/POST) with Axios and React Router

**Symfony Examples:**
- Full-stack task management application with CRUD operations
- Docker containerization with FrankenPHP
- Doctrine ORM and database migrations

### 🎯 Practice Assignments

Apply what you've learned through structured assignments that reinforce course concepts.
